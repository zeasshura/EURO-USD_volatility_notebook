# EUR/USD: предсказуем ли валютный курс?

Сравнение ARIMA, Ridge/ElasticNet, Random Forest и CatBoost на задаче прогнозирования
дневной доходности EUR/USD — с проверкой гипотезы эффективного рынка (EMH) и
кластеризации волатильности на реальных данных ЕЦБ.

## TL;DR

- **Знак** дневной доходности $r_t$ **не предсказывается** ни одной из 4 моделей (R² ≤ 0 везде) — рынок эффективен.
- **Амплитуда** доходности $|r_t|$ (волатильность), наоборот, **предсказывается** стабильно лучше константного бейзлайна: R² от +0.023 до +0.055, ROC-AUC от 0.598 до 0.620, статистически значимо (p < 10⁻⁵).
- Бэктест сигнальной стратегии на прогнозе знака подтверждает вывод: после издержек стратегия не обыгрывает buy-and-hold.

## Почему это не очередной "прогноз курса валют"

- **Честная валидация.** Хронологический train/test split (80/20, без перемешивания) + `TimeSeriesSplit(gap=5)` внутри train — исключены заглядывания в будущее через скользящие признаки.
- **Честные бейзлайны.** Модели сравниваются не сами с собой, а со случайным блужданием и медианным прогнозом.
- **Статистическая строгость.** ADF-тест на стационарность рядов, z-тест на значимость ROC-AUC — не просто "метрика выросла", а проверка, что это не шум.
- **Экономическая интерпретация.** Помимо ML-метрик — бэктест с Sharpe и MaxDD, чтобы показать, что значит "предсказуемость" в деньгах (и что она почти ничего не значит после издержек).

## Данные

Официальные дневные референсные курсы евро от ЕЦБ — [Euro foreign exchange reference rates](https://data.ecb.europa.eu), файл `euro-daily-hist_1999_2022.csv`, период 1999–2022, 29 валют.

## Методология

1. **Признаки** (~80 шт.): лаги log-доходностей, скользящие средние/EMA, MACD, RSI, полосы Боллинджера, реализованная волатильность на нескольких окнах, первые 5 компонент PCA по кросс-валютным доходностям (PCA обучен только на train — без утечки).
2. **Модели**: ARIMA (линейная авторегрессия) → Ridge/ElasticNet (регуляризованная линейная регрессия) → Random Forest → CatBoost — от простого к сложному, с сопоставимой валидацией на каждом шаге.
3. **Валидация**: 5-fold `TimeSeriesSplit(gap=5)` для подбора гиперпараметров, финальная однократная оценка на отложенном тесте.
4. **Метрики**: RMSE / MAE / R² для регрессии, DirAcc / ROC-AUC для бинарной интерпретации (знак / превышение медианы), Sharpe / MaxDD для экономической интерпретации.

## Результаты

| Модель | R² на r (знак) | R² на \|r\| (волатильность) | ROC-AUC на \|r\| |
|---|---|---|---|
| Baseline (RW / медиана) | 0 | 0 | 0.50 |
| Ridge | ≤ 0 | +0.023 | 0.598 |
| ElasticNet | ≤ 0 | +0.039 | 0.615 |
| Random Forest | ≤ 0 | **+0.055** | 0.620 |
| CatBoost | ≤ 0 | +0.051 | 0.620 |

Все модели независимо приходят к одному и тому же контрасту: знак непредсказуем,
амплитуда — предсказуема. Это согласуется с классической теорией (Fama 1970 для
первого момента, Mandelbrot 1963 / Engle 1982 — для второго), а не противоречит ей.

```bash
pip install pandas numpy scikit-learn statsmodels catboost matplotlib scipy
```

## Ограничения

Это research-проект, а не production-сервис: нет API, деплоя, мониторинга дрифта
данных. Эффект на волатильности статистически значим, но невелик по абсолютной
величине (R² ~0.02–0.05) — это ожидаемо для сравнительно эффективного рынка, а не
недоработка модели.
